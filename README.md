> **Your AI now has a face, emotions, ears, voice, eyes вҖ” and a soul.**  
> 100% local inference. Zero telemetry. Zero cloud. Just vibes.

![ProxyFace Landing Page вҖ” dark mode with Preteen character selected](docs/screenshots/1_landing_dark.png)

ProxyFace renders a pixel-art avatar that reacts in **real time** to LLM output via a 4 MB TinyBERT emotion model running entirely on your GPU (WebGPU) or CPU (WASM). It listens, speaks, watches your eyes, and never sends a single byte of your conversation anywhere.

## вңЁ What makes it special

### рҹҺ§ Hands-Free (HF) вҖ” learn languages while you talk
Hold **Alt+T** to speak. Release to send. The AI replies in your target language with its face reacting to every word вҖ” embarrassed, curious, delighted. No typing. No clicking. Just conversation.

> *"I use it to practice Japanese. The pumpkin face going SURPRISED every time I say something wrong is weirdly motivating."*

### рҹ§  4 MB emotion brain вҖ” runs at 60 ms on your GPU
TinyBERT INT8 ONNX, trained on 3 200 sentences across 8 emotions. Runs via WebGPU in Chrome вҖ” no Python, no server, no API key for inference. The face reacts to the AI's output, not yours.

### рҹҺЁ 40+ pixel-art characters вҖ” or submit your own
![Character gallery вҖ” Dev, Daruma, Console, Terminator](docs/screenshots/3_characters.png)

Drop a sprite sheet in `sprites/art/yourname/` and run one sync script. Your character appears instantly. [Submit it to us](#submit-your-art) for priority review and a place in the official gallery.

### рҹ‘ҒпёҸ Eye tracking вҖ” opt-in, on-device
MediaPipe face landmarker runs locally. The pupils follow your gaze. No video ever leaves your machine.

### рҹ”Ҡ Voice I/O вҖ” bot mode, natural mode, or silent
- **Bot mode**: typewriter sound while the AI streams
- **Natural mode**: browser TTS reads the reply aloud
- **HF mode**: hold Alt+T вҶ’ speak вҶ’ auto-send вҶ’ AI replies

### рҹ”’ Privacy-proof
Zero network calls for inference. Your API key lives in `localStorage`, never transmitted to us. GPL-3.0 вҖ” read every line.

## рҹҡҖ Quickstart

### Web (browser, any OS)
```bash
git clone https://github.com/PacifAIst/Proxyface.git
cd Proxyface
pnpm install
cd apps/web && pnpm dev
# open http://localhost:5173
```

### Windows desktop app
Download **[ProxyFace Setup 0.1.0.exe](https://github.com/PacifAIst/Proxyface/releases/latest)** from Releases.  
> Windows SmartScreen may appear вҖ” click **More info вҶ’ Run anyway**. This is expected for unsigned indie apps.

### Mock mode (no API key needed)
Visit `http://localhost:5173/?mock=1` вҖ” uses a regex classifier instead of the neural model. Good for UI testing.
